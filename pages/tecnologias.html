TECNOLOGIAS UTILIZADAS

A FloodWatch emprega um sistema integrado que combina hardware e software para monitoramento e alerta de enchentes. Tecnologias principais: Sensores Inteligentes (IoT): Instalação de sensores de nível de água em pontos estratégicos, como rios e bueiros. Esses sensores capturam dados em tempo real e os atualizam no sistema a cada segundo. Sistema de Monitoramento e Análise de Dados: Uma plataforma central que recebe e processa as informações coletadas pelos sensores. Comunicação e Notificação: As informações são enviadas automaticamente para órgãos públicos e disponibilizadas em um site informativo acessível à população. O sistema emite alertas automáticos via web e notificações para a população local em caso de risco iminente de enchente. Interface do Usuário: Uma interface simples e direta para o usuário final, e uma interface adaptável a dispositivos móveis (RNF02).
